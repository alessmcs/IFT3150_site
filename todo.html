Idées de verbalisation de fonctions lexicales

Source intéressante: Normalisation des FL

  • Anti: quel est l'antonyme de __. Plusieurs types d'antonyme. Pour le mot x:
    • contradictoire (non x) (ex. non présent -> absent)
      • Comment dit-on non-x?
    • contraire: n'est pas x (ex. n'est pas ouvert -> fermé)
      • Comment dit-on que quelque chose n'est pas x?
    • scalaire: plus ou moins. Par exemple, quel est le mot pour dire 'est moins lourd?' -> léger
      • Quel est le mot pour dire [+/-] x?
    • position opposée. Ex. Quelle est la position opposée à gauche? -> droite
  • Conv: difficile à généraliser
  • Gener: généraliser. Quel est un concept général dont le mot x fait partie? Qu'est ce qu'un x en un mot?
  • Figur: terme figuratif. Donner une [figure de style/métaphore] utilisant le mot x.
  • Contr: terme contrastif, lexies cohabitent de façon contrastive dans des phrases, utilisés pour des fins stylistiques
    • Quel est le terme qui entre en contraste avec x?
  • S0: nominalisation. Donner le nom commun associé au mot x. Quel est le nom correspondant à x?
  • V0: verbalisation. Quel est le verbe correspondant à x?
  • A0 adjectivisation, Av0 adverbialisation. Même principe.
  • Claus clausativisation: Que dit-on pour signaler [un/le fait de] x?
  • Dérivations de noms d'actants & de circonstants
    • S1. Qui ou qu'est-ce qui fait l'action de x? Qui ou qu'est-ce qui agit sur x?
    • S2. Qu'est ce qui est visé /traité/? par l'action de x? (ex S2(vendre)=marchandise)
    • S3. Qui ou qu'est ce qui reçoit les conséquences de l'action de x? (ex S3(vendre) = acheteur)
    • S4. S4(vendre) = prix ?? Maybe pcq on vend à un certain prix? Mm logique pour aller [quelque part]?? idk
    • Equip: Comment nomme-t-on l'équipe qui fait fonctionner x?
    • Cap: Comment nomme-t-on le chef de x?
    • S_instr: Quel instrument est employé pour réaliser l'action de x?
    • S_med: Que consomme-t-on/utilise-t-on pour réaliser l'action de x? Qu'est-ce qui est consommé lorsqu'on réalise l'action de x?
    • Sloc: Dans quel lieu est fait x?
  • Syn (peut-être): Quel est le synonyme [+- riche] de x? Quel est un synonyme [familier/soutenu] de x?
    • Moins trivial que de simplement demander de donner un synonyme?
